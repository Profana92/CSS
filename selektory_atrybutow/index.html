Podstrona ta poświęcona jest zrozumieniu selektorów atrybutów. Zwyczajne selektory klas oraz atrybutów nie stanowią większego problemu w ich zrozumieniu, jednak zawansowane koncepcje pierwszeństwa reguł stanowią problem bardziej złożony, którego analizie służy ta podstrona. Najpierw rozważmy element p, o przypisanym kolorze liniowym niebieskim i stylu css: p { color: red; } p[id] { color: red !important; }
